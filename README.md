SAE – Développement d’un Dashboard
Analyse de données du CEM Rennes

------------------------------------------------------------

CONTEXTE

Ce projet est réalisé dans le cadre d’une SAE (Situation d’Apprentissage et d’Évaluation)
par des étudiants en Sciences des Données.

L’objectif est de concevoir un dashboard analytique à partir des données du CEM Rennes,
en réponse à une demande d’un client fictif composé de docteurs et de statisticiens.

Ce dashboard a vocation à servir de modèle d’inspiration pour la création future
de tableaux de bord destinés à des professionnels de santé travaillant sur des données réelles.

------------------------------------------------------------

OBJECTIFS DU PROJET

Objectifs pédagogiques :
- Mettre en pratique les compétences en traitement et analyse de données
- Appliquer des méthodes statistiques adaptées
- Concevoir des visualisations pertinentes et rigoureuses
- Traduire un besoin métier en solution analytique

Objectifs fonctionnels :
- Concevoir un dashboard clair et exploitable
- Mettre en avant des indicateurs clés pertinents
- Faciliter l’interprétation des données
- Proposer une structure adaptable à des données médicales réelles

------------------------------------------------------------

PUBLIC CIBLE

Client supposé :
- Docteurs
- Statisticiens
- Professionnels de santé

Le dashboard doit être :
- Scientifiquement rigoureux
- Clair et structuré
- Adapté à un public expert
- Orienté aide à la décision

------------------------------------------------------------

STRUCTURE DU DEPOT

/data        : Jeux de données utilisés dans le cadre du projet
/src         : Scripts de traitement et d’analyse
/dashboard   : Fichiers liés à la conception du dashboard
/assets      : Ressources visuelles
README.txt   : Documentation du projet

------------------------------------------------------------

DONNEES

Les données utilisées dans ce projet sont exploitées exclusivement
à des fins pédagogiques.

Aucune donnée sensible n’est diffusée publiquement dans ce dépôt.

------------------------------------------------------------

FINALITE

Le dashboard développé constitue :
- Un prototype fonctionnel
- Une preuve de concept
- Une base méthodologique pour la création future de dashboards médicaux professionnels
